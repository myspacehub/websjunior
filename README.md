# 福建省初中全科学习 H5

独立本地目录：`outputs/fujian-junior-high-h5`

包含文件：

- `index.html`：页面入口
- `styles.css`：移动端自适应样式
- `app.js`：课程数据、思维导图、学习方法、B站资源与交互逻辑

功能：

- 九大学科目录：语文、数学、英语、道德与法治、历史、地理、生物、物理、化学
- 七八九年级分册与章节导航
- 每章知识思维导图
- 每科跨册总图
- 初中学习方法模块
- B站分档 UP 推荐
- B站章节搜索：只使用章节标题本身作为关键词，避免搜索词过长
- 手机端自适应

本地预览：

```bash
cd outputs/fujian-junior-high-h5
python3 -m http.server 41740
```

然后打开：

```text
http://127.0.0.1:41740/
```

说明：

课程框架按福建初中常用教材体系整理；不同学校若存在教材版本差异，可直接在 `app.js` 的 `COURSE` 数据中调整分册和章节。
