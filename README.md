# 福建泉州初中全科学习 H5（2026现行教材版）

独立本地目录：`outputs/fujian-junior-high-h5`

包含文件：

- `index.html`：页面入口
- `styles.css`：移动端自适应样式
- `app.js`：课程数据、思维导图、学习方法、B站资源与交互逻辑

功能：

- 九大学科目录：语文、数学、英语、道德与法治、历史、地理、生物、物理、化学
- 泉州全市统一教材版本：语文/历史/道法部编版，数学华东师大版，英语仁爱科普版，地理人教版，生物北师大版，物理沪科版，化学沪教版
- 七八九年级分册与章节导航
- 每章知识思维导图
- 每科跨册总图
- 初中学习方法模块
- B站分档 UP 推荐
- B站章节搜索：泉州版本学科自动带教材关键词，例如“华东师大版初中数学 二次函数”“仁爱版初中英语 被动语态”“沪科版初中物理 电功率综合计算”
- 手机端自适应

本地预览：

```bash
cd outputs/fujian-junior-high-h5
python3 -m http.server 41740
```

然后打开：

```text
http://127.0.0.1:41740/
```

泉州学习节奏：

- 七年级：全科打底，语数英主科打牢；地理、生物同步学习，建立会考基础
- 八年级：新增物理；地生全年冲刺初二会考；数学几何函数、英语语法进入难点
- 九年级：新增化学；地生结业后不再学；重点攻九上二次函数、电学、酸碱盐、道法国情、近现代史

说明：

课程框架按福建泉州 2026 现行统一教材体系整理，全市统一、无区县差异。若后续学校实际用书发生调整，可直接在 `app.js` 的 `COURSE` 数据中调整分册和章节。
